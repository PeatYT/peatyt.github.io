Visualization 1: UFO Sightings Over Time

Writeup

I am visualizing the sightings of UFOs plotted against the year in which they were seen. I've used line marks and made the color green to better match the UFO theme. I was unable to get the years to line up nicely to the data, I think because some of the years were missing from the data. I've transformed the data by removing the most recent year since its data is incomplete, along with dropping NA values from the dataset. Additionally, I've recast the year column to be a date type.

Visualization 2

Writeup

This visualization id of the shapes of UFOs and their frequency, sorted via interactivity by state. The color is still green to be UFO themed. This visualization required additional data cleaning to be tolerant of errors, with dropping of NA values in the state column to avoid null states available in the dropdown. This is the plot that contains interactivity. For interactivity, I've made a dropdown that lets the viewer select a state and dynamically changes the available columns and values to match the choice. The visualization changes corresponding to the choice, with both the column sizes and the presence of columns adapting to what is present in the chosen state's data. I've chosen not to limit the dropdown to US states, instead leaving it global since it lets users see the difference across multiple countries if they were interested in such information.
